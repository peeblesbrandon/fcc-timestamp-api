API Project: Timestamp Microservice

By Brandon Peebles for freeCodeCamp

API endpoint usage:

  • The API endpoint is GET https://timestamp-microservice-peebles.glitch.me/api/timestamp/:date_string?
  • Date string must be valid (e.g. parseable by new Date(date_string))
  • An empty date string will return the current timestamp
